Introdução ao HTML na prática

(Voltar)

O módulo "Introdução ao HTML na Prática" apresenta os conceitos básicos para criar páginas web usando HTML. O conteúdo inicial explica a estrutura básica de um documento HTML, destacando a declaração !DOCTYPE html e as tags html, head e body, que organizam as informações complementares e o conteúdo principal.

Na prática, são exploradas tags importantes como títulos (h1 a h6), parágrafos (p), links (a), listas ordenadas e não ordenadas, além da tag (br) para quebras de linha e melhor organização visual. O módulo também aborda atributos genéricos como id, para identificar elementos de forma única; class, para agrupar elementos com estilos ou comportamentos semelhantes; e style, que permite aplicar estilos diretamente no HTML. A compreensão desses elementos é fundamental para manter o código organizado e facilitar a integração com CSS e JavaScript, fornecendo uma base sólida para o desenvolvimento web e o avanço em tecnologias front-end.

Trabalhando com formulários em HTML

(Voltar)

O módulo "Trabalhando com Formulários em HTML", aborda os principais conceitos para a criação de formulários em páginas web. São apresentados os elementos fundamentais, como a tag (form) e seus atributos action e method, além dos campos de entrada (input, textarea, label, button) e seus diferentes tipos (texto, senha, e-mail, número, data, entre outros).
Também são explorados atributos importantes como name, placeholder, required e disabled, que controlam o comportamento e a validação dos campos. O módulo destaca ainda boas práticas de estruturação e acessibilidade, preparando o aluno para criar formulários funcionais, organizados e compatíveis com futuras integrações em aplicações web.

Estruturando seu HTML + formatações

(Voltar)

O módulo "Estruturando seu HTML + Formatações" apresenta técnicas para organizar o código HTML de forma clara e eficiente, além de introduzir diversas tags de formatação de texto, como (strong, mark, b, i, u, font, blockquote e fieldset). Também explica a diferença entre as tags (embed e iframe), usadas para incorporar conteúdos externos, e aborda os principais formatos de especificação de cores, incluindo nomes, hexadecimal, HSL, RGB e RGBA, etc. O módulo destaca ainda o uso das tags (div e span) para agrupar conteúdo e facilitar a estilização via CSS, consolidando uma base sólida para o desenvolvimento front-end.

Trabalhando com mídias utilizando HTML

(Voltar)

O módulo "Trabalhando com Mídias utilizando HTML" apresenta o uso das tags (img, svg, audio, video, track e iframe) para incorporar diferentes mídias em páginas web. Explica os formatos de imagem JPG, PNG e SVG, destacando suas características e diferenças. O conteúdo aborda como inserir imagens, gráficos vetoriais, áudios, vídeos e legendas, além de incorporar conteúdos externos por meio do iframe. Esse conhecimento é essencial para criar páginas web mais dinâmicas, acessíveis e interativas.

Criando tabelas com HMTL

(Voltar)

O módulo "Criando Tabelas com HTML" ensina a estruturar dados tabulares em páginas web usando as tags (table, caption, thead, tbody, tfoot, tr, th e td). O conteúdo aborda como criar a tabela, adicionar um título descritivo com (caption), definir seções distintas para cabeçalho, corpo e rodapé com (thead, tbody e tfoot), e organizar linhas e células com (tr, th e td). O módulo destaca a importância dessas tags para organizar, formatar e apresentar dados de forma clara, legível e acessível em sites.

Entendendo HTML Semântico

(Voltar)

O módulo "Entendendo HTML Semântico" explica a importância de usar tags semânticas como (header, nav, main, section, article, aside e footer) para organizar o conteúdo de forma lógica e acessível. Destaca a diferença entre HTML4 e HTML5, ressaltando que o HTML5 introduziu essas tags e trouxe melhorias para multimídia, formulários e APIs. O módulo também aborda o conceito de web scraping, ressaltando que uma estrutura semântica clara facilita a extração automatizada de dados por ferramentas especializadas. Além disso, explica que SEO (Search Engine Optimization) é o conjunto de práticas para melhorar a visibilidade e o posicionamento de sites nos resultados dos motores de busca, e que o uso correto do HTML semântico contribui significativamente para isso, assim como para a acessibilidade dos sites.
